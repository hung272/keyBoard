# keyBoard

English version below

一個自己寫的全局鍵盤處理

把 AppDelegate 裡面的

extension AppDelegate
extension UIResponder 
extension UIApplication

 複製到您的專案
 
 然後在 AppDelegate didFinishLaunchingWithOptions
 
 寫上 keyBoardHandleStart()
 
 搞定
 
 恭喜您，已經完成APP全局的 鍵盤事件處理了
 
 覺得不錯的話，給個星星吧！～
